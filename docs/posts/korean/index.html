Thanks to all the previous comments I managed to get it to work with audio. I thought I'd write up how I installed it as in case helps anyone.

The following instructions were for a clean install of Anki version 2.0.32 on Windows 7.

1. In Anki, go to >tools >browse & install and enter the code 1959202519 then close Anki
2. Go to the folder C:\Users\YOUR_USER_NAME_HERE\Documents\Anki\addons directory and delete the folder Memrise_Course_Importer and the file MemriseCourseImporter.py
3. Go to github here https://github.com/wilddom/memrise2anki-extension and click the 'download zip' button and save the zip file to the desktop.
4. Extract the zip file to your desktop.
5. Copy the contents of the memrise2anki-extension-master folder to C:\Users\YOUR_USER_NAME_HERE\Documents\Anki\addons (you should end up with a folder called Memrise_Course_Importer in the addons with 2 other files MemriseCourseImporter.py and README.md.
6. Open anki and go to >tools >import memrise course and enter your memrise login details.
7. Then paste the link of the course you want to download (I used http://www.memrise.com/course/2086/the-450-most-common-french-verbs/ ) and select the relvant options from the dialogue (I left them at their default settings).
8. Wait for the files to download.
9. Under assign memrise fields leave the options as suggested
